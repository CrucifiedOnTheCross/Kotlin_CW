# Система учета клиентов автомойки

Консольное приложение на языке Kotlin для курсовой работы по теме 17.

## Возможности

- загрузка клиентов из `car_wash_clients.json`;
- просмотр всех записей;
- добавление клиента;
- редактирование клиента по ID;
- удаление клиента по ID;
- поиск по ФИО, телефону, госномеру и модели автомобиля;
- сортировка по ФИО, сумме оплат, количеству посещений и дате визита;
- расчет агрегированных показателей;
- сохранение данных в JSON-файл;
- обработка некорректного пользовательского ввода.

## Структура проекта

- `src/main/kotlin/model` - модели данных;
- `src/main/kotlin/service` - бизнес-логика и работа с JSON;
- `src/main/kotlin/ui` - консольное меню, ввод и вывод;
- `src/main/kotlin/Main.kt` - точка входа;
- `car_wash_clients.json` - демонстрационные данные.

## Запуск

```powershell
.\gradlew.bat run
```

## Сборка

```powershell
.\gradlew.bat clean test installDist
```

После сборки исполняемый файл будет доступен в:

```text
build/install/car-wash-client-accounting/bin/
```
